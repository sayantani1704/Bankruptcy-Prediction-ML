# Company Bankruptcy Prediction using Machine Learning

This project analyzes financial data of companies and applies machine learning models to predict whether a company is likely to go bankrupt.

## Tools & Technologies
Python  
Pandas  
NumPy  
Scikit-learn  
Matplotlib  

## Machine Learning Models Used
Logistic Regression  
Random Forest  
Gradient Boosting  

## Project Workflow
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature selection
- Model training and evaluation
- Prediction of bankruptcy risk

## Project Goals
Identify financial patterns associated with bankruptcy  
Build predictive models to classify companies as bankrupt or non-bankrupt  
Help businesses and investors make data-driven decisions

## Files Included
bankruptcy_analysis.ipynb – data analysis notebook  
dataset.csv – financial dataset  
model.pkl – trained machine learning model  
project_report.pdf – detailed project documentation
